# MovieCatalog – личный каталог фильмов и сериалов

## Цель проекта
Создать простое веб-приложение для ведения личного каталога просмотренных (и не только) фильмов и сериалов. Пользователь сможет добавлять, просматривать, оценивать и удалять записи.

## Функциональность
- Добавление фильма/сериала с полями: название, год выпуска, жанр (драма, комедия и т.д.), моя оценка (1–10), статус («Просмотрено» / «В планах»).
- Просмотр списка всех добавленных карточек.
- Удаление ненужных записей.
- (В будущем) поиск и фильтрация по жанру/статусу.

## Предполагаемая архитектура
- **Backend**: Node.js + Express
- **Хранилище**: SQLite
- **Frontend**: простой HTML + CSS + JavaScript 
- **Запуск**:
  - `npm install`
  - `npm start` – запускает сервер на `http://localhost:3000`
